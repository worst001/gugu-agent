# Design Specification

## I. Project
- Title: Gugu Agent 产品与架构介绍
- Format: PPT 16:9, 1280 x 720
- Language: Chinese
- Page count: 8

## II. Audience
- 技术负责人、开发者、潜在使用者和合作方。
- 目标是在 8-12 分钟内讲清产品定位、核心能力、系统架构和近期方向。

## III. Style Objective
- 产品路演 + 技术架构简报。
- 视觉基调稳重、现代、可信，避免营销感过强。
- 使用原生矢量形状、文本框、流程图、矩阵和路线图，保持 PowerPoint 可编辑。

## IV. Color Scheme
- ink: #1D2329
- muted: #667085
- paper: #F7F8FA
- white: #FFFFFF
- rust: #8F482F
- teal: #2D628F
- olive: #4F6237
- amber: #F4C95D
- sand: #F3E7DA
- blue_soft: #D9E8F6
- green_soft: #E2EBDD
- rose_soft: #F4DAD4
- line: #D8DEE6

## V. Typography
- Font family: Microsoft YaHei
- Cover title: 56 px
- H1: 38 px
- H2: 28 px
- Body: 22 px
- Caption: 16 px
- Code / labels: 18 px

## VI. Page Outline
1. 01_cover: Gugu Agent, 本地可运行的 Claude Code
2. 02_positioning: 产品定位与一句话价值
3. 03_user_value: 为什么需要它
4. 04_architecture: 桌面端到 CLI 的三层架构
5. 05_agent_skills: 多 Agent 与 Skills 引擎
6. 06_desktop_im: 桌面端与 IM 远程工作流
7. 07_computer_use_security: Computer Use 与安全控制
8. 08_roadmap: 近期路线与收束

## VII. Image Resource List
- No external bitmap image is required.
- All visuals are authored as SVG-native diagrams and editable shapes.

## VIII. Execution Notes
- Use top-level SVG groups for animation-friendly structure.
- Avoid filters, marker arrows, foreignObject, CSS gradients, and unsupported SVG features.
- Use separate text elements for line breaks.
