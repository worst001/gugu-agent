# Gugu Agent v0.1.15

这版集中处理小白用户首次使用、长任务稳定性、文件交互和发布流程。也补齐了任务完成通知，方便长任务结束后及时回到桌面端查看结果。

## 体验优化

- 任务完成后弹出系统通知，Windows 和 macOS 均会给出完成提示，并加入应用内提示音兜底。
- 计划模式改为会话内确认，计划确认后自动回到默认模式，避免多会话弹窗混乱。
- 任务执行中锁定权限和模式切换，减少运行中改配置导致卡住的问题。
- 修复首次提交新会话后消息列表空白，需要切换会话才恢复的问题。
- 修复魔法棒生成提示词后无法回滚的问题。
- 优化网页搜索展示，合并网页搜索次数，减少不必要的状态干扰。
- 统一 Gugu / Gugu Desktop 身份文案，减少底层 SDK 名称露出。

## 文件与工具

- 文件路径点击可打开系统文件浏览器，Windows 打开资源管理器，macOS 打开 Finder。
- 支持从文件浏览器拖拽文件或图片到聊天框。
- 对大文件、压缩包和 GLM 文件解析增加保护：可本地解压的压缩包优先本地处理，不能解压的不再走 Gugu Managed/GLM。
- 附件解析支持更多可恢复错误提示，GLM parser 结果处理更稳。
- 内置 codegraph 与 rtk 能力，方便构建代码知识图谱并节省 token。

## 稳定性与发布

- 增强 Agent 断线和恢复提示，减少长任务无响应造成的误解。
- 自动化桌面发布流程：GitHub Actions 构建 Windows MSI、macOS Apple Silicon DMG，并上传 updater 产物、OSS 和 Gitee Release。
- 修复 macOS updater manifest 生成和构建脚本换行问题。
- Gateway 拆为子项目，并更新下载页和下载安全说明。
