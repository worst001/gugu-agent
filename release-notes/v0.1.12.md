# Gugu Agent v0.1.12

This release refreshes the Windows package so a clean install includes the default agent plugin pack out of the box.

## Changes

- Bundled the default `claude-mem`, `engineering-skills`, and `engineering-advanced-skills` plugins into the packaged agent resources.
- Added those plugins to the built-in `gugu-bundled` marketplace manifest so first launch can register and enable them through the normal plugin flow.
- Extended the bundled agent pack regression test to verify packaged plugins are installed and enabled.
