# Gugu Agent v0.1.12

本次发布更新了 Windows 安装包，干净安装后即可开箱使用默认 Agent 插件包。

## 更新内容

- 将默认的 `claude-mem`、`engineering-skills` 和 `engineering-advanced-skills` 插件打包进 Agent 资源。
- 将这些插件加入内置的 `gugu-bundled` marketplace 清单，首次启动时会通过正常插件流程完成注册和启用。
- 扩展了内置 Agent 包的回归测试，验证打包后的插件可以正确安装并启用。

## 验证
