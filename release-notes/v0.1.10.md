# Gugu Agent v0.1.10

这是一个体验修复版本，重点处理桌面端对话流结束状态、工具结果展示、提示词优化与会话恢复稳定性。

## 主要更新

- 优化桌面端流式会话状态：当模型返回澄清、普通文本或自然结束消息时，输入框会正确恢复可编辑状态，避免按钮停留在“停止”。
- 改进工具调用与工具结果展示，减少长内容、结构化结果和空结果在聊天列表中的视觉抖动。
- 优化工作区变更识别逻辑，让最近修改内容的选择更贴近真实编辑结果。
- 提示词优化支持按原始语言重试，减少中英文场景下的语言漂移。

## 体验优化

- 思考过程、工具块和流式状态的显示更紧凑，聊天记录在运行中更容易浏览。
- 计划确认、计划更新和本地回滚流程的前端状态更稳定。
- 空会话和 CE 工作流相关文案做了统一调整。

## 问题修复

- 修复部分文本回复已经完成但输入框仍显示“停止”的问题。
- 修复会话恢复、附件解析和会话历史在边界情况下可能出现状态不同步的问题。
- 修复提示词优化失败后重试时可能没有正确保留语言偏好的问题。

## 安装说明

### macOS

首次打开如果提示“已损坏”或“无法验证开发者”，请执行：

```bash
xattr -cr /Applications/Gugu\ Agent.app
```

### Windows

下载 `.msi` 安装包后直接安装即可。如果安装前已经运行旧版本，请先退出托盘中的 Gugu Agent 再安装。
