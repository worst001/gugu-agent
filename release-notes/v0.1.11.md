# Gugu Agent v0.1.11

这是一个发布准备版本，重点完善托管额度、购买入口、官网下载页、CE 模式控制和桌面端稳定性。

## 主要更新

- 新增 Gateway 官网首页和下载页，购买页改为套餐选择与人工订单流程，为后续微信、支付宝支付接入预留订单基础。
- 新增 Gateway 管理看板能力，可查看设备、套餐、订单、用量和下载配置，方便运营统计。
- 桌面端订阅页改为更轻量的套餐与用量百分比展示，隐藏点数细节，保留购买和激活入口。
- Agent 模式控件改为默认 / 计划 / CE 的折叠主控，CE 模式下再展示工作流选择，工具栏更简洁。
- README、桌面截图和架构图更新为 Gugu Agent 品牌素材，移除旧项目捐赠和无关品牌内容。

## 体验优化

- 关于页改为商业化入口：展示官网与下载、官方渠道和 Gitee 问题反馈，不再突出源码仓库。
- 默认模式下减少无意义的本地“思考中”占位，普通聊天会更快显示真实回复状态。
- 购买页套餐排版、订单复制和选择状态更清楚。
- 网关下载配置支持版本号和 SHA256，后台可直接查看当前发布包信息。

## 问题修复

- 修复 Gugu Managed 上游配置异常时流式请求可能长期卡住的问题。
- 修复 WebSocket 断连重连窗口过短导致长时间思考任务被提前停止的问题。
- 修复普通文本 `message_stop` 过早结束服务端 turn monitor 的问题。

## 安装说明

### macOS

首次打开如果提示“已损坏”或“无法验证开发者”，请执行：

```bash
xattr -cr /Applications/Gugu\ Agent.app
```

### Windows

下载 `.msi` 安装包后直接安装即可。如果安装前已经运行旧版本，请先退出托盘中的 Gugu Agent 再安装。
