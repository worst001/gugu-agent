# Gugu Agent v0.1.17

本次版本聚焦 Gugu Managed 稳定性、长任务可见性、macOS 启动可靠性，以及支付网关发布前的生产加固。

## 修复

- 修复 macOS 打包版识别不到 `gugu-sidecar` 的问题，避免会话启动时错误回退到不存在的 `/bin/claude-gugu`。
- 修复大文件 `Write(content)` 后续请求继续携带完整文件内容的问题，降低上下文膨胀和网关 500 风险。
- 订阅额度用完时，桌面端改为展示友好的 Gugu 额度提示卡，不再直接暴露 CLI 启动失败错误。
- 长任务执行时会显示正在运行的工具名和耗时；超过 30 秒会提示任务仍在运行，可继续等待或停止后缩小任务范围。

## 改进

- Gateway 增加消息、附件解析、下单、设备注册等入口的限流、排队、超限保护和维护模式写冻结。
- Gateway 增加 MySQL 存储实现、SQLite 到 MySQL cutover/rehearsal 工具，以及 post-cutover 支付订单监控脚本。
- Gateway 支付通知幂等性增强，微信和支付宝通知会按交易号去重并保护订单状态。
- 桌面包继续内置 `.agents/skills` 能力包，包含 skills、bundled plugins 和 plugin-provided agents。

## 验证

- Desktop chat 相关测试、lint、build 已通过。
- Gateway 全量测试已通过。
- 生产 MySQL cutover 后，微信和支付宝真实支付订单均已履约并发码；24 小时 post-cutover 监控无异常。
