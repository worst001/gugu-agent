# Gugu Agent v0.1.9

这是一个面向上线发布的整理版本，重点补齐订阅中心、本地激活校验、桌面端会话标题稳定性，以及 Gugu Agent / 谷星曜工作室的公开展示。

相比 `v0.1.8`，本次把 billing 能力先落在本地边界：桌面端会显示订阅状态、购买入口和激活码输入，后端只返回本地 mask 后的信息与最近校验结果，不会在未配置真实购买/校验地址时发起外部网络请求。

## 主要更新

- 设置页新增「订阅」入口，展示当前计划、到期时间、最近校验时间和本机激活码后四位，便于上线前灰度验证订阅流程。
- 新增本地 billing API 与桌面端 store：支持读取订阅状态、打开购买地址、提交激活码并触发校验；未配置真实 URL 时保持本地安全降级。
- 会话标题生成进一步稳定：补齐服务端标题生成测试与桌面端标题兜底，避免空标题或重复默认标题影响多会话识别。
- 附件解析能力增强，提升粘贴/拖拽内容进入服务端后的结构化解析稳定性。
- README、文档首页、安装说明和相关资源链接改为 Gitee 发布入口，并统一使用 Gugu Agent 品牌。
- 关于页作者与社交媒体展示统一改为「谷星曜工作室」。

## 体验优化

- 去掉 CE 工作流角色下拉框的阴影，让顶部工具栏视觉更轻、更贴近当前暗色界面。
- 桌面端标签栏与工作区模型展示做了细节修整，减少标题、会话与模型状态不同步时的视觉跳动。
- 更新说明、仓库链接和安装文档从旧 GitHub 项目迁移到当前 Gitee 仓库，避免用户下载时走到旧地址。

## 问题修复

- 修复 billing 相关接口在没有远程配置时可能误触发网络校验的风险，现在会明确返回本地开放状态。
- 修复附件解析中部分边界格式可能导致服务端识别不完整的问题。
- 修复部分会话在标题生成或会话恢复阶段状态漂移的问题，让桌面端列表更稳定地保留真实会话标题。

## 安装说明

### macOS

首次打开如果提示“已损坏”或“无法验证开发者”，请执行：

```bash
xattr -cr /Applications/Gugu\ Agent.app
```

### Windows

下载 `.msi` 安装包后直接安装即可。如果安装前已经运行旧版本，请先退出托盘中的 Gugu Agent 再安装。
