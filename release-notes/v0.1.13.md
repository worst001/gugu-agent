# Gugu Agent v0.1.13

这个版本重点整理桌面端发布体验、订阅与能力入口，并把默认 agent / plugin / skill 资源随安装包一起带上。支付宝后端仍可在 gateway 侧继续迭代，不阻塞本次桌面端打包发布。

## 主要更新

- 桌面端随包内置默认能力包 `gugu-agent-pack`，包含常用 agents、插件和 skills；安装后无需额外同步这些默认资源即可在能力面板、Agents、Skills、Plugins 页面查看。
- 订阅与能力面板更清晰：能力栏补齐终端入口，订阅入口保留在设置区，减少重复导航。
- IM 接入面向国内应用收敛，只展示飞书、钉钉、企业微信和 QQ；本地诊断与保存流程不再把 Telegram 作为主入口。
- 购买页、结算页和下载页做了轻量视觉整理，订单支付区域与套餐选择拆开，Windows / macOS 下载入口更清晰。
- 公开页面和桌面端文案进一步统一为 Gugu / 谷星曜共创社，减少旧品牌露出。

## 支付与官网

- 微信 Native 支付链路已经接入 gateway：创建订单、展示二维码、轮询支付状态、支付成功后自动生成激活码。
- Dashboard 补充支付渠道、交易号、金额和支付时间等信息，仍保留人工收款、发码、取消兜底操作。
- 下载地址切换为新的 OSS 命名规范，后续版本按 `Gugu-Agent-<version>-windows-x64.msi` 和 `Gugu-Agent-<version>-aarch64.dmg` 延续。

## 体验优化

- IM 接入页的“检查配置”改为更轻的胶囊按钮，避免窄宽度下文字挤压换行。
- Computer Use、权限弹窗、空会话、输入框等桌面端文案统一改为 Gugu。
- 下载页安全说明改为低压力提示区，降低未代码签名阶段的用户心理负担。
