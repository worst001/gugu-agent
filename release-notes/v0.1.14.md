# Gugu Agent v0.1.14

本次更新重点优化桌面端新用户的首次使用体验、附件处理，以及更安全的本地工作流。

## 主要更新

- 聊天输入框支持拖拽文件，包括从 Windows 文件资源管理器和 macOS Finder 拖入文件。
- 优化本地路径点击处理，Agent 输出中的文件或目录路径可以更可靠地在资源管理器或 Finder 中打开。
- 优化首次启动时的能力统计，内置 agents、插件和 skills 会立即显示数量，不再先显示为 0。
- 将默认 Gugu agent pack 打入桌面端资源包，包含 agents、skills 和内置插件。
- 增加更安全的本地 ZIP 处理：支持的 ZIP 压缩包会解压到当前项目的 `.gugu/archive-extracts/` 下并在本地分析，不再上传到 Gugu Managed 或 GLM。
- 增加附件大小限制和大文件保护，避免用户上传超大文件时导致桌面端或 gateway 卡死。
- 优化 Computer Use 新手引导和环境安装提示，帮助缺少 Python 或运行依赖的用户完成配置。
- 优化 WebSearch 展示，补充中文标签，并合并显示网页搜索次数。
- 调整托管模型默认配置，让默认体验更偏向稳定的深度思考能力。
- 优化长任务中的重连和 Agent 卡住恢复逻辑。
- 降低工具错误提示的视觉噪音，并优化本地历史记录加载失败时的提示。
- 优化更新检查逻辑，在发布元数据缺失或过期时给出更友好的反馈。

## 支付与 Gateway

- 保持微信和支付宝收银台流程与当前 gateway 实现兼容。
- 为 gateway 附件解析增加请求大小保护。
- 托管附件解析继续只处理小文件；较大的本地文件或压缩文件会留在本地处理。
