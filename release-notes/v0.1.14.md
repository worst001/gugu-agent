# Gugu Agent v0.1.14

This release focuses on first-run reliability, attachment handling, and safer local workflows for new desktop users.

## Highlights

- Added drag-and-drop file support for the chat composer, including files dropped from Windows Explorer and macOS Finder.
- Added clickable local path handling so agent output paths can open in Explorer or Finder more reliably.
- Improved first-run capability counts so bundled agents, plugins, and skills show up immediately instead of appearing as zero at startup.
- Bundled the default Gugu agent pack into the desktop app resource package, including agents, skills, and bundled plugins.
- Added safer local ZIP handling: supported ZIP archives are extracted into the current project under `.gugu/archive-extracts/` and analyzed locally instead of being uploaded to Gugu Managed or GLM.
- Added attachment limits and large-file guards to avoid freezing the desktop app or gateway when users upload very large files.
- Improved Computer Use onboarding and environment guidance for users without Python or runtime dependencies.
- Improved WebSearch display, Chinese labels, and grouped search count presentation.
- Updated managed model defaults to favor stronger reasoning for the default experience.
- Improved reconnect and stalled-agent recovery behavior for long-running sessions.
- Softened noisy tool error UI and improved local history reload failure messaging.
- Improved update-check behavior when release metadata is missing or stale.

## Payment And Gateway

- Kept WeChat and Alipay checkout flows compatible with the current gateway implementation.
- Added request-size protections for gateway attachment parsing.
- Kept managed attachment parsing small-file only; larger or compressed local files stay local.
