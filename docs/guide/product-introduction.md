# Gugu Agent 产品介绍

Gugu Agent 是一款运行在本地电脑上的 AI 工作助手，帮助你更轻松地写代码、改代码、整理资料、处理文档和完成自动化任务。

你可以把它理解成一个懂项目、会操作工具、能记住上下文的智能助手。它既可以在终端里使用，也可以通过桌面端像普通聊天软件一样使用。

## 为什么要使用 Gugu Agent

- 本地可运行：可以在自己的电脑上启动，直接处理当前项目和本地文件。
- 写代码更省心：可以阅读代码、解释逻辑、修改文件、展示 Diff，并帮你排查问题。
- 桌面端更直观：提供图形化界面，支持多会话、多标签、文件搜索、附件上传和工具调用展示。
- 支持多种模型：可以接入 DeepSeek、OpenAI、OpenRouter、Ollama、GLM 等兼容模型，按团队或个人需要灵活选择。
- 附件也能处理：支持上传图片、PDF、Word、Excel、PPT 等资料，让 AI 先读懂内容，再帮你总结、改写或分析。
- 会记住重要信息：记忆系统可以保存用户习惯、项目背景和常用规则，后续对话不用反复说明。
- 可以分工协作：多 Agent 系统可以把复杂任务拆给不同助手并行处理，例如一个读代码、一个写方案、一个做检查。
- 能使用专业技能：Skills 系统可以把固定流程做成可复用能力，例如代码审查、调试、测试、发布检查等。
- 可通过聊天软件使用：支持 Telegram、飞书、钉钉、企业微信、QQ 等 IM 接入，出门在外也能和项目里的 AI 助手沟通。
- 可以操作电脑：Computer Use 能让 AI 在授权后截屏、点击、输入和切换应用，帮助完成一些需要界面操作的任务。
- 权限可控：涉及改文件、执行命令、操作电脑等动作时，可以先确认再执行，避免误操作。

## 适合谁使用

- 研发同事：写功能、修 Bug、读源码、补测试、整理技术方案。
- 产品和项目同事：写需求、拆任务、整理会议纪要、总结项目进展。
- 运营和市场同事：改写文案、生成表格、分析材料、整理客户信息。
- 客服和支持同事：总结问题、生成回复话术、沉淀常见问答。
- 团队管理员：统一配置模型、插件、技能和远程 IM 使用方式。

## 可以用来做什么

### 写代码和改代码

你可以直接告诉它：

```text
帮我看一下这个报错原因，并修改相关代码。
```

它会读取项目、定位问题、提出改法，并在需要时修改文件。

### 阅读和总结资料

你可以上传文档或粘贴内容，然后说：

```text
请帮我总结核心结论、风险点和下一步行动。
```

它可以把长资料整理成更容易阅读的结构。

### 拆解复杂任务

你可以把一个比较大的目标交给它：

```text
帮我把这个功能拆成开发计划，并列出需要验证的点。
```

它可以生成步骤、任务清单和注意事项。

### 远程处理项目

配置 IM 接入后，你可以在聊天软件里给 Bot 发消息，让它创建会话、继续任务、停止生成或处理权限确认。

## 第一次使用建议

1. 先安装并打开 Gugu Agent。
2. 在设置里配置一个主聊天模型，例如 DeepSeek。
3. 如果要上传图片、PDF 或 Office 文件，再配置 GLM 文件与图片解析。
4. 先从简单任务开始，例如总结一份文档、解释一段代码、改写一段话。
5. 需要改代码或执行命令时，先使用默认的询问权限模式。

## 一句话总结

Gugu Agent 让 AI 不只是陪你聊天，而是能真正进入你的工作环境，理解项目、处理文件、调用工具，并在你确认后帮你完成具体工作。
